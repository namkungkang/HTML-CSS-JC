Box Model
콘텐츠와 바깥쪽에 있는 테두리 사이의 간격은 padding
테두리와 테두리 사이는 마진값

즉 컨텐츠를 축으로 보더를 만들면 안엔 패딩 바깥쪽엔 마진이라 생각하면 쉬움

★★★Box Model은 부피감을 결정한다는 것이고 디자인에서 핵심

Block vs Inline
화면 전체를 쓰는 것은 Block
자기 크기만 갖는 것을 inline
컨텐츠의 크기를 정할 때 폭은 width, 높이는 height 값을 지정하면 됨

Box Sizing
ex) 같은 컨텐츠에 보더 크기를 다르게 하고, width 값을 같게 지정해주면 엘리먼트의 크기가 다르다 
이유는 컨텐츠영역의 크기를 width 값으로 지정했기 때문에 컨텐츠의 크기는 같지만 보더 값이 다르기에 엘리먼트 크기가 다르다
★★★그래서 Box Sizing을 통해서 엘리먼트의 크기를 같게 해준다 
content-box 기본으로 지정

border-box 크기를 지정할 때 보더를 기준으로 지정할 수 있게 해서 엘리먼트의 크기를 같게 해준다
